## How to Run Locally

1️⃣ **Clone the Repository**  
```bash
git clone https://github.com/ryuu9505/my-portfolio.git
```

2️⃣ **Navigate to the Project Directory**  
```bash
cd my-portfolio
```

3️⃣ **Install Dependencies**  
```bash
npm install
```

4️⃣ **Run the Project**  
```bash
npm start
```
